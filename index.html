The Workflow

1. Define Prompt

Add descriptive prompts directly to your img tags using standard data attributes.

2. Execute Agent

Run the Python script. It parses your HTML, generates images via DALL-E, and saves them locally.

3. Auto-Injection

The agent updates your HTML source, swapping prompts for path-mapped local files instantly.

How to Use MarkupVision

1. Add Markup

Place prompts directly in your HTML image tag using data-prompt.

2. Run File

Run the Python script and let it generate image assets from your prompt descriptions.

3. Auto Replace

The script rewrites src paths so the markup points to the newly generated local files.
